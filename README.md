# Boost_Converter-12V-24V_5A-
Simulation of Boost converter in LTspice which converts 12V to 24V with current throughput of 5A.
## Components Used
1.5uH Inductor\
104.66uF Capacitor\
N-Channel Mosfet(BSZ0905NS)\
4.8 Ohms(Load)
